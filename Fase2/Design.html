Propósito del Sitio

El próposito del sitio es de nivel educativo, donde por medio de la utilización de este sitio web
los estudiantes puedan ingresar desde sus casas y complementen su aprendizaje con temas de desarrollo web
tratando parte teórica por medio de articulos y contenido escrito, como también de forma visual, apoyandose
de elementos visuales seleccionados y construidos para absorver conocimientos de forma práctica.

Flex Box

En esta área se explicará y dará una guía de como funciona el diseño de CSS Flex Box,
se verá sobre los conceptos básicos y sus propiedades, terminología, las propiedades
del contenedor flexible, como es su sintaxis, entre muchos otros conceptos sobre este tema.

CSS Grid

En esta área se explicará y dará una guía de como funciona el CSS Grid,
como realizar diseños utilizando esta herramienta, como se divide una página
en áreas o regiones principales, como se define la relación en términos de tamaño,
posición y capas entre partes de un control construido a partir de primitivas HTML,
entre otros aspectos de este tema.
